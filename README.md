## 🔹 Core Stata Package Features  

<p align="center">

Each package focuses on solving real-world research & survey data challenges. 🚀  
</p>

<div align="center">

<tr>

  <li>✅ Counts missing/error codes <code>-99, 99, -999</code></li>
